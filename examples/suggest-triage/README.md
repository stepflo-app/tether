# suggest-triage (coming soon)

This is a roadmap placeholder for v0.1.x.

The plan: a Linear issue is created, an agent suggests 2–3 candidate teams using
context from the issue body and historical assignments, and Slack posts a
single-message ballot to `#triage`. An engineer clicks one option; if nobody
clicks within the timeout, the suggestion rejects.

Tracked separately from the v0.1 walking skeleton. See the repo issues for
status.
